CashVest is an application created for IST 261 at Penn State University

The repostiory for this can be found on Github

Mark Tripoli
Author

JavaFX

CashVest was meant to be created as a fully Java-based application. The UI of choice for this project is JavaFX using OpenJFX 11

OpenJFX

Spring Framework

To power the "back-end" portion of this application, I have chosen the Java Spring Framework. This is due to its ideas of Inversion of Control (IoC) and Dependency Injection (DI). Frankly, I just wanted to make a desktop app with it....
